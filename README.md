# Study Flutter

This repository is dedicated to my daily Flutter and Dart training.

Here, I experiment with:
- Dart algorithm practice (loops, lists, maps, conditions)
- Manual Flutter widget writing (Navigator, BlocBuilder, TextField...)
- UI rendering and navigation exercises
- Katas: progressive exercises to master Clean Architecture and state management

---

## Folder Structure

| Folder/File              | Description                                       |
|--------------------------|---------------------------------------------------|
| `lib/katas/`             | Progressive katas (Cubit, Clean Architecture)     |
| `lib/dart/`              | Dart algorithm exercises                          |
| `lib/widgets/`           | Manual widget writing from memory                 |
| `lib/features/`          | Feature-based exercises (Tag, Todo)               |
| `flutter_katas_plan.md`  | Full training program (12 katas + quizzes)         |

---

## Why this repo?

Even though it's not a production app, this repo is part of my real learning process.
Everything pushed here represents effort and growth — it's my training ground to sharpen my Flutter and Dart skills.
